Tic Tac Toe game

Problem statement

In a 4*4 grid, two players to take turns with X and O respectively. The first person to consequently get 3 symbols in a row wins.

Assumptions:

  • Player Vs Player.
  • First player starts with X.

Pseudocode

Steps:
  1. Start the program by clicking the start button.
  2. Make the grid visible after clicking start. X is the first move.
  3. Alternatively X and O should be entered in the cells by clicking.
  4. Next move to be shown at the side.
  5. For every click, check the adjacent cells, if 3 consecutive X or O, game ends. Else game to be continued.
