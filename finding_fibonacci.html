Finding the fibonacci of a given number

Problem statement

Input a number from the user. Output the fibonacci series of that number.

Assumptions:

  • Number must be positive integers.

Pseudocode

Steps:
  1. Start the program
  2. Input the number from the user
  3. Find the fibonacci of that number by the following steps
  4. Formula: Given the number N, fibonacci(N) is the sum of the fibonacci of two previous numbers.
  5. first find fibonacci of 1, which is 1.
  6. Find fibonacci of 2, which is sum of fibonacci of previous 2 nos. i.e., (0+1) = 1
  7. repeat the steps until the given input
  8. Output the result.
