Finding the factorial of a given number

Problem statement

Input a number from the user. Output the factorial of that number.

Assumptions:

  • Number must be positive integers.

Pseudocode

Steps:
  1. Start the program
  2. Input the number from the user
  3. Find the factorial of that number by the following steps
  4. Formula: Given the number N, N! = N*N-1*N-2*.. *1
  5. Output the result.
