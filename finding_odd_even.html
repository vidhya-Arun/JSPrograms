Finding odd or even numbers

Problem statement

Input 10 numbers from the user and find if they are odd or even and print the result along with the numbers

Assumptions:

  • Numbers must be positive integers.
  • Input exactly 10 numbers in the range of 1-100

Pseudocode

Steps:
  1. Start the program
  2. Input the number from the user
  3. Store the number
  4. Repeat steps 2 and 3 ten times
  5. Take each number and find the modulus of 2
  6. If the result is 0, print it as even number along with the number. Else print it as Odd number
  7. Repeat steps 5 and 6 10 times
