Finding the duplicate number count

Problem Statement

Input the 10 numbers from the user. Compare the numbers and find the count of duplicate among them and display it along with the problem statement, assumptions, pseudocode, scratch program link and the javascript program.

Assumptions:

  • Numbers must be positive integers.
  • Input 10 numbers in the range of 1-100

Pseudocode

Steps:
  1. Start the program
  2. Input 10 numbers from the user
  3. store the numbers in an array.
  4. Make 2 for loops, compare first array element with the rest of the elements.
  5. If the numbers match, increment the duplicate counter and break from the loop.
  6. Repeat steps 4 and 5 and display the duplicate count .
